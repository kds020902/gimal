# 안나가공 컨트롤러

Flutter로 만든 오버레이 기반 컨트롤러 앱입니다.  
메인 앱에서 [URL 검색, 북마크, 메모]를 관리하고,
오버레이 창을 통해 다른 화면 위에서도
웹 검색과 메모 확인을 할 수 있습니다.

## 화면 예시

![앱 메인 메뉴 화면](docs/screenshots/overlay_launcher.png)

## 주요 기능

- URL 또는 검색어를 입력해 Google 검색 및 웹사이트 접속 및 북마크 저장 
- 저장된 북마크 목록 확인 및 수정
- 메모 추가, 수정, 삭제
- 오버레이 전환 버튼을 통해 플로팅 아이콘 실행
- 오버레이 안에서 URL 검색, 북마크 열기, 메모 관리

- 북마크와 메모 관리 화면 UI/UX 정리
- 코드 최적화로 APK 용량 줄이기
- 오버레이 사용 중, 화면을 내리더라도 데이터 저장
- 앱 아이콘과 전체 디자인 완성도 높이기
